This website is the first step in a multi-part project for developing web based applications. In its current state it is a static website built with a responsive template, future versions will be built on a framework with server-side capabilities to allow for dynamic content.

This project has a deeper purpose beyond boasting about my schooling and credentials. The container this website was built off acts as a centralized point to route visitors to projects hosted under different sub domains. In addition to duties as a reverse proxy, behind the scenes it is managing ssl certificates. The certificate manager renews certificates before they expire and can issue new ones as more subdomains are needed. The limiting factor, frankly, is the host, with the containerized implementation and structure of my project redeploying on different infrastructure will merely take an afternoon.

Stay posted to this website as I will begin documenting and hosting projects that I have been talking about and designing for years. These projects will include:

Mail Server

I will host a container that will manage emails under the connorlynch.ca domain. This project will be an opportunity to learn more about mail servers and how they interact with reverse proxies. The container should also have some type of programmatic interface to allow applications to send and receive emails, such as…

Land Collection

This web app would act as a visual inventory for game pieces I collect in the trading card game; Magic the Gathering. Currently I have a mongoDB database with triggers to import new cards from an API and sort them into collections that correspond to my personal library. The first version will be very standard organizing, searching and displaying. It will be extended to include more features that I’ll outline in its own document.
